Usage: cli [OPTIONS] COMMAND [ARGS]...

Commands:
  housekeeping  container for miscellaneous house keeping commands
  projects      List code projects found in the directory
  reporters     container for reporter commands
  test          Eze run scan
  test-online   Eze run scan remotely on a server
  test-remote   Eze run scan against git repo, and report back to...
  tools         container for tool commands
