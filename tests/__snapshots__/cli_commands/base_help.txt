Usage: cli [OPTIONS] COMMAND [ARGS]...

  Eze Command line interface

Options:
  --version   Show the version and exit.
  -h, --help  Show this message and exit.

Commands:
  housekeeping  container for miscellaneous house keeping commands
  projects      List code projects found in the directory
  reporters     container for reporter commands
  test          Eze run scan
  test-online   Makes a call to the api to run eze scan remotely on a server
  test-remote   Pull a remote repository from git and run eze scan, report...
  tools         container for tool commands
